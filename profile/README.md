# S E C U R I T Y — I N T E R N A T I O N A L — G R O U P

A global team across four nations, advancing secure Unix software.

## Our Purpose

We build trustworthy Unix software. Security is not a feature—it is the foundation.

We believe in duty, integrity, clarity, and resilience. We do not compromise. We do not hide.

## Our Principles

**Security** We choose the harder path.

**Transparency** Our code is reviewed, our decisions documented.

**Simplicity** Fewer moving parts, each well-understood.

**Thought** We build for decades, not quarters.

**Collective** No person is above scrutiny or praise. 

## Our Work

- Secure Unix software development and maintenance
- Collaborative code review across regions
- Shared knowledge and best practices
- Audited, transparent systems

## Four Regions

- **Region 1** — Infrastructure and compliance
- **Region 2** — Cryptographic research
- **Region 3** — Kernel-level security
- **Region 4** — Deployment and hardening

Each operates autonomously, accountable to the whole.
